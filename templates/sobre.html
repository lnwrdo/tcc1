SOBRE O PROJETO

Visão Geral do Projeto

Este projeto foi desenvolvido com o objetivo de criar uma plataforma interativa para recomendações de filmes. A interface oferece aos usuários uma experiência imersiva e intuitiva, permitindo a navegação entre diferentes seções e uma escolha de filmes personalizada.

Utilizando tecnologias modernas de desenvolvimento web, a aplicação integra estilos visuais atraentes com uma arquitetura robusta, garantindo fluidez e desempenho.
